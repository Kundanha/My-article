Contains Duplicate

Array mein duplicate elements dhundne ke 4 tarike ‚Äî simple se optimized tak! üöÄ

üü¢ Easy Level

Problem Kya Hai?

Ek integer array nums diya gaya hai. Tumhe batana hai ki array mein koi duplicate value hai ya nahi. Agar koi bhi value ek se zyada baar aati hai, toh true return karo, warna false.

üéØ Example: [1, 2, 3, 1] ‚Üí true (kyunki 1 do baar aa raha hai)
üéØ Example: [1, 2, 3, 4] ‚Üí false (sab unique hain)
1

Brute Force Approach

Intuition (Sochne Ka Tarika)

Sabse simple aur seedha tarika ‚Äî har ek element ko baaki saare elements ke saath compare kar lo! üîç

Matlab agar array mein 5 elements hain, toh pehle element ko baaki 4 se compare karo, phir doosre ko baaki 3 se, aur aisa hi chalte jao. Agar kabhi bhi do same values mil gayi, toh duplicate mil gaya!

üí≠ Socho jaise class mein sabke birthdays check kar rahe ho ‚Äî har student ko har doosre student se poochna padega!

Algorithm (Steps)

  1. Do nested loops lagao ‚Äî outer loop i ke liye, inner loop j ke liye
  2. Inner loop i + 1 se start karo (khud se compare mat karo!)
  3. Agar nums[i] == nums[j] toh turant true return karo
  4. Saari pairs check hone ke baad koi match nahi mila? false return karo
Python 
class Solution:
    def hasDuplicate(self, nums: List[int]) -> bool:
        for i in range(len(nums)):
            for j in range(i + 1, len(nums)):
                if nums[i] == nums[j]:
                    return True
        return False

Sorting Approach

Intuition (Sochne Ka Tarika)

Agar array ko sort kar dein, toh jo bhi duplicate elements honge, wo ek doosre ke paas aa jayenge! üìä

Phir bas ek simple loop lagao aur har element ko uske pichle wale se compare karo. Agar dono same hain = duplicate found! ‚úÖ

üí≠ Jaise students ko height ke hisaab se line mein khada karo ‚Äî same height wale automatically paas paas aa jayenge!

Algorithm (Steps)

  1. Array ko sort karo (chhote se bade)
  2. Index 1 se loop start karo (0 se nahi, kyunki pehle element ka previous nahi hota)
  3. Har element ko uske previous element se compare karo
  4. Agar dono equal hain ‚Üí true return karo
  5. Loop khatam ho gaya bina match ke? ‚Üí false
Python 
class Solution:
    def hasDuplicate(self, nums: List[int]) -> bool:
        nums.sort()  # Array ko sort kar do
        for i in range(1, len(nums)):
            if nums[i] == nums[i - 1]:
                return True
        return False

O(1)

Hash Set Approach

Intuition (Sochne Ka Tarika)

Hash Set ek aisi data structure hai jo sirf unique values store karti hai, aur kisi bhi value ko O(1) time mein dhundh sakti hai! üî•

Idea simple hai: Ek empty set lo. Array mein ghoomte jao ‚Äî agar current number pehle se set mein hai, toh duplicate mil gaya! Warna use set mein daal do.

üí≠ Socho jaise ek register maintain kar rahe ho ‚Äî har naya naam aane pe pehle check karo kahin pehle se toh nahi likha, agar hai toh duplicate, warna likh do!

Algorithm (Steps)

  1. Ek empty set banao (seen = set())
  2. Array ke har number pe loop lagao
  3. Check karo: kya ye number pehle se set mein hai?
  4. Agar hai ‚Üí true return (duplicate mila!)
  5. Nahi hai ‚Üí set mein add kar do
  6. Loop khatam, koi duplicate nahi ‚Üí false
Python 
class Solution:
    def hasDuplicate(self, nums: List[int]) -> bool:
        seen = set()  # Empty set banao
        for num in nums:
            if num in seen:  # Pehle dekha hai?
                return True
            seen.add(num)  # Set mein daal do
        return False

Time & Space Complexity

‚è±Ô∏è Time Complexity
O(n)
üíæ Space Complexity
O(n)
üöÄ Best time complexity! Har element ko sirf ek baar dekhte hain. Lekin extra space lagti hai set ke liye ‚Äî classic time-space tradeoff!

Hash Set Length (One-Liner) üéØ

Intuition (Sochne Ka Tarika)

Ye previous approach ka shortcut version hai! Set automatically duplicates remove kar deta hai.

Toh agar original array ki length set ki length se zyada hai, iska matlab kuch elements remove ho gaye ‚Äî yani duplicates the! üß†

üí≠ Jaise 10 students ki class mein agar unique roll numbers 8 hi nikle, toh 2 duplicates zaroor hain!

Algorithm (Steps)

  1. Array ko set mein convert karo (duplicates auto-remove)
  2. Set ki length ko array ki length se compare karo
  3. Agar set chhota hai ‚Üí true (duplicates the!)
  4. Dono barabar ‚Üí false (sab unique the)
Python 
class Solution:
    def hasDuplicate(self, nums: List[int]) -> bool:
        return len(set(nums)) < len(nums)

Time & Space Complexity

‚è±Ô∏è Time Complexity
O(n)
üíæ Space Complexity
O(n)

Common Galtiyan (Pitfalls)

Galat Comparison in Brute Force

Nested loops mein sabse common mistake ‚Äî inner loop ko i se start karna instead of i + 1. Isse element khud se compare ho jata hai!

Python ‚ùå Wrong 
# Element khud se compare ho raha hai!
for i in range(len(nums)):
    for j in range(len(nums)):
        if nums[i] == nums[j]:
            return True
Python ‚úÖ Correct 
# i+1 se start = no self-compare
for i in range(len(nums)):
    for j in range(i + 1, len(nums)):
        if nums[i] == nums[j]:
            return True

Original Array Modify Karna

Sorting approach mein nums.sort() use karne se original array badal jata hai! Agar original order chahiye, toh copy bana ke sort karo.

Python ‚ö†Ô∏è Risky 
# Original array badal jayega!
nums.sort()
Python ‚úÖ Safe 
# Copy pe sort = original safe
sorted_nums = sorted(nums)
